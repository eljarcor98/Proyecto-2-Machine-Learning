# ⚽ Proyecto 2: Machine Learning - Premier League

Este proyecto se enfoca en el análisis y modelado de datos de la Premier League, basado en los requerimientos del taller 2.

## 📂 Estructura del Proyecto

- **data/**: Contiene los datos crudos (`raw`) y procesados (`processed`).
- **notebooks/**: Jupyter Notebooks para análisis exploratorio (EDA) y experimentación.
- **src/**: Scripts de Python para limpieza de datos, entrenamiento de modelos y utilidades.
- **models/**: Modelos entrenados guardados (formatos .pkl, .h5, etc.).
- **reports/**: Figuras y reportes generados.

## 🚀 Inicio Rápido

1. Instalar dependencias:
   ```bash
   pip install -r requirements.txt
   ```
2. Explorar los notebooks en la carpeta `notebooks/`.
